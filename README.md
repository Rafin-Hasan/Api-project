## Installation
1. Clone the repository:

    ```sh
    git clone https://github.com/yourusername/custom-api-website.git
    ```
2. Navigate to the project directory:

2. Navigate to the project directory:

    ```sh
    cd custom-api-website
    ```
3. Install dependencies:
  ```sh
    cd custom-api-website
    ```
3. Install dependencies:
 ```sh
    npm install
    ```
